

# Adicionar "Instituto de Pesquisa" na lista de Tipos de Instituicao

## Alteracao

Adicionar a opcao `{ value: "instituto_pesquisa", label: "Instituto de Pesquisa" }` ao array `INSTITUTION_TYPES` no arquivo `src/components/my-account/InstitutionCombobox.tsx`.

A lista passara de 4 para 5 opcoes:
- Instituicao de Ensino
- Empresa
- ONG
- Orgao Publico
- **Instituto de Pesquisa** (novo)
